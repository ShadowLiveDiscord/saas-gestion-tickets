# 🎫 TicketFlow — SaaS Gestion de Tickets

Plateforme de gestion de tickets de support client en HTML/CSS/JS pur, sans backend.

## 🚀 Pages disponibles

| Page | Description |
|------|-------------|
| `index.html` | Landing page publique |
| `pages/login.html` | Connexion |
| `pages/register.html` | Inscription |
| `pages/dashboard-user.html` | Espace utilisateur |
| `pages/dashboard-agent.html` | Espace agent |
| `pages/dashboard-admin.html` | Panel administrateur |

## 🛠️ Fonctionnalités

- **3 rôles distincts** : Utilisateur, Agent, Administrateur
- **Gestion des tickets** : création, suivi, filtres, priorités, statuts
- **Messagerie interne** sur chaque ticket
- **Panel admin** : KPIs, gestion utilisateurs/agents, paramètres
- **100% frontend** : fonctionne sans serveur (localStorage)
- **Design moderne** : responsive, dark-ready
