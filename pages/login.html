Configurer Google Sign-In

Un Client ID Google est requis

Comment obtenir un Client ID :

  1. Allez sur console.cloud.google.com
  2. Créez un projet → APIs & Services → Credentials
  3. Cliquez Create Credentials → OAuth 2.0 Client ID
  4. Type : Web application
  5. Ajoutez dans "Origines autorisées" : http://localhost:3000
  6. Copiez le Client ID généré
